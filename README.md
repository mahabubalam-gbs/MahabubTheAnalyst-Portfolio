# MahabubTheAnalyst-Portfolio
🧩 Data Analytics Portfolio

A results-driven data analytics portfolio showcasing hands-on projects in EDA, SQL analysis, web scraping, and interactive dashboards, demonstrating strong skills in data cleaning, analysis, visualization, and business insights.

📂 Projects
1. Hotel Booking EDA (Python)

Explored hotel booking datasets and performed data cleaning and feature analysis.

Created visualizations to identify trends and patterns.

Delivered actionable business insights for hotel operations.

2. Telco Customer Churn Analysis (Python)

Analyzed customer churn datasets to identify key drivers of churn.

Visualized churn patterns across different segments.

Provided insights to improve customer retention strategies.

3. E-commerce Data Analysis (SQL & Python)

Performed SQL queries and Python analysis on e-commerce datasets.

Examined sales, revenue, and customer behavior.

Produced insights for operational and strategic decisions.

4. Web Scraping of Top U.S. Companies (Python)

Scraped data from top U.S. company websites.

Structured and processed the data for analysis.

Demonstrated automation and data collection skills.

5. BlinkIT Sales Analytics Dashboard (Power BI)

Built an interactive Power BI dashboard for sales performance and KPIs.

Analyzed outlet distribution, item categories, and business trends.

Enabled data-driven decision-making.
